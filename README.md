# 🎯 Guess the Number Game

A simple command-line game in Python where you try to guess a randomly generated number between **1 and 1000**.

---

## 🚀 Features

- 🔢 Random number generation (1–1000)  
- 📈 Feedback on whether your guess is too high or too low  
- 🔁 Option to play again after winning  
- 🆘 Help section (triggered by typing `9999`)  
- 🚫 Handles invalid, non-integer, and out-of-range inputs  
- 👨‍💻 Beginner-friendly code structure  

---

## ⚙️ How It Works

- 📝 Enter your name and begin guessing  
- 🎮 Guess a number — the game tells you if it’s higher or lower  
- 🏆 When you guess correctly, you're told how many tries it took  
- 🔁 You can choose to play again or quit  
- 📖 Type `9999` at any time to view the help section

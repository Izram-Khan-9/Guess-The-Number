# 🎮 Guess The Number

A fun Python-based number guessing game with 5 difficulty levels, a scoring system (Diamond, Ruby, Emerald, Gold, Silver, Bronze), and random hints when you’re close to the hidden number.

🔢 Difficulty levels: Easy → Impossible
🏆 Scoring system: Diamond → Bronze
🎲 Randomized hints for close guesses
⌛ Simple animations for a fun experience
